四年的本科生活一晃而过，在这里写点东西给学弟学妹们，也算是总结一下自己对计算机学科的一些观察与感悟。

### 1. 计算机专业概览
计算机科学与技术专业，可以理解为去学习与电脑相关的周边产品的原理以及理论。借用徐朝农老师的一句话，我们培养的学生不是去做PS设计的，而是去制作PS软件的。关于计算机，你需要知道下面这些事：
- 首先，大家选择这个行业，是极其幸运又颇具挑战的
- 电脑坏了，学计算机专业的不会修很正常
- 搞计算机理论研究的，不是整天在电脑前，而是一支笔加一张纸研究各种符号
- 学计算机的并不累，各行各业想做好都需要投入精力
- 学计算机不拼爹，除非你创业拿融资
- 帝都已经形成计算机和非计算机两大专业，阶级已经产生，你去看工资，看他们的生活状态，是不是很美好
- 计算机是各行各业的工具，其培养人的算法思维方式，不同于单纯实验性质或背的学科
- 计算机各个方向有相关性，但对找工作而言不大，你只需要专一个方向，这也是非科班狂转计算机的可行性之一
- “开源”这个词，可以蒙骗很多雇主
- 你喜欢机器人？现在计算机占机器人知识结构的70%，其余才是机械和电路，原因在于SOC板上系统技术，例如你看到的树莓派等。
- 当某个人给你看很牛的程序时，很可能他是直接从网上找的，这是明显的信息不对称有没有？
- 计算机和数学很配哦，去搜“老顾谈几何”公众号看看，学长告诉你，数学知识真的是越多越好，越深刻越好
- 计算机的图书挺好写的，博客也很好写
- 一般互联网公司的薪酬，普通开发<算法<系统架构

### 2. 大学培养的计算机人才和从北大青鸟等出来的专才的区别

如果高校出去的毕业生做的工作同样是JAVA WEB开发，ANDROID开发等，如果大学期间较颓废，差别可能真不大，学生还要到公司再培训几个月才能上岗。但是什么时候会体现出你大学毕业的优势呢？那就是遇到一些系统层面，算法层面的问题，当然这种情况出现的频率也不高，尤其对于中小公司，单纯做开发的那种。搞开发需要具备什么技能呢？  
- 积木搭的好。以做一个网站为例，捯饬JAVA安装，环境变量配置，纠结CLASSPATH不存在；eclipse安装，甚至是myeclipse安装，然后还要破解；安装Tomcat服务器，纠结各种安装方式，配置方式，目录结构；上网搜Myeclipse结合Tomcat服务器开发知识；安装Mysql数据库，安装失败，纠结怎么关闭端口和服务，再次尝试安装；安装Sqlyog，纠结连接方式与使用，学sql语法；安装Java的数据库JDBC驱动包，纠结包是否奶瓶状才能用；从网上找连接数据库测试代码，测试一下；写一个HTML页面，包含form表单，纠结POST和GET方式提交区别；纠结页面太丑，添加CSS样式；添加JAVA WEB相应的jsp特性，纠结中文字符显示问题；纠结前端和后端数据传送方式；部署到云服务器，纠结代码怎么上传，GIT or SVN，项目如何打包，云应用如何配置，包括数据库连接；开始浏览器访问。

- 捯饬，好奇心。尝试装各种软件，尝试不同的操作系统，开发语言与框架，立志做一名全栈开发工程师。
- 电脑配置不要太差
- 互联网信息检索的能力，大部分搞开发的都是在找现有的解决方案
- 整理自己的知识库，代码库
- 对相关的技术有整体全面的认识，你最好知道图书馆4楼计算机图书的体系

### 3.你想搞科研
计算机方向的学术文章几乎都是免费的，你可以在arxiv上第一时间获得最前沿的学术成果，你可以了解一下RSS订阅功能，尽管有很多灌水文章。你可能觉得科研很难，但我认为就问题的难度而言，它比高数简单多了，会出现很多专业词汇来吓唬你，并且是英文的，有很多领域内约定俗成的叫法，这会让初涉者摸不着头脑。要有兴趣督促自己，投入足够的时间都可以做的很好。  

搞深度学习方向某一领域的研究流程：查看网上各种博文，知道大概后再查看原始论文，有代码的话查看github代码，部署环境，下载数据集训练，分析代码。

### 4.关于各个课程

- 语言类的课程：在学过C语言后几乎都可以自学，自学最好的方法是github找对应语言的学习代码，部署环境，分析运行代码。面向对象语言C++和JAVA了解封装继承多态的概念，以及命名空间，包等概念，可以多个语言对比学习。学习C++，你可以去看OPENCV和CAFFE的实现。
- 数据结构：要整理代码库，重视实现，再复习一下ACM，看剑指OFFER图书
- 操作系统：实验是在linux下运行的，学习linux命令行操作，cmake，make，vim等，终会受用；也研究一下ROS机器人操作系统。
- 计算机组成原理：结合数字逻辑，用FPGA实现一个CPU，看CPU设计图书。阅读Atmega的数据手册，ARM9数据手册，了解微处理器架构
- 计算机网络：自己尝试组网试试，假设自己开了一个公司
- 机器学习：下载python源码并分析，让数学成为你的朋友，有兴趣的看看泛函分析，复变函数，矩阵论，信息论，随机过程等，数学知识真的越多越好，有时候觉得现在深度学习调网络结构的工作真的很幼稚，需要我们更加理性一些
- 编译原理：从事后诸葛的角度看，阅读开源编译器源码是不错的选择。
- 软件工程：概括性比较高的学科，我们一般开发总是纠结于技术本身，远未达到其所说的高度，了解即可


### 5.谈谈实习
我大二下学期暑假的时候曾受吴双元老师推荐到一家医疗设备公司做软件开发，实际在那边工作了近一个月，当然在没有去那边之前在学校这边远程的也做了些工作。这次经历让我第一次体会到朝九晚五的生活是什么样子，每天早上我一个人骑着自行车去那边，20分钟左右。从公司大门进去要刷指纹，我有好几次去晚了，自己怪不好意思地低着头迅速走到工位开机。。。我不太习惯这个公司的某些文化，它要求所有员工身穿灰色的那种制服，还要求工作期间上衣拉链必须拉到最上面然后翻转领口，大夏天的，我当时还是有些不适应，但也可能是自己娇生惯养了吧！公司里有人力资源管理的，他们做的工作无非就是员工们的吃喝拉撒，当时我们着急工作，领着我们的那个人力资源管理的女的非要带我们学习董事长的创业经历以及企业的发展历史，商业定位等等，在我们再三强调我们与其他的员工不同时，那人才罢休！我也注意到一个中年男子正打算签劳动协议，发现必须工作3年以上，于是打了退堂鼓，3年，那人脸上的无奈，失落，我似乎从我父亲当年的脸上看到过。我相对开放，和一起参与软件开发的其他人相处的很快，与我一同实习的还有王蕊，我们专业的另一位女生，但是她很内向，被动型的说话，我觉得这将近一个月的实习对她来说是一种折磨。吴老师在头几次与我们一起去那边和他们那边的工程师交流项目，吴老师每次说话不卑不亢，声音洪亮有力，指引着交流的进程，这对我们来说是一种鼓舞，那时候觉得吴老师特别帅！


项目的内容是做煎药管理系统，而我负责物流管理模块，就是学习使用顺丰的开发接口，然后对接他们的系统。系统采用.net框架，c#语言编程，数据库采用sqlserver2008，项目部署在公司的服务器上，工具软件公司有整理，用win7自带的内网进行文件分享。.net虽然low，但是采用其技术的公司还是比较多的，这些工程师甚至不懂何谓css，可能勾选图形界面习惯了。外包公司一般都有自己的代码库，比如访问数据库的通用代码，而这些代码在csdn上通常都可以下载到。在涉及实际项目时，可能要设计类，属性和方法的访问权限用private还是public？怎么访问？似乎自己说的算。static型的变量又在什么情况下使用呢？大家真正做的时候是不是要舍近求远呢，必须要get()和set()设置类的属性吗？可能我的理解不深刻吧！我在做项目的时候，大部分时间在浏览网页，查csdn，国外博客。我记得我要打印顺丰的快递单，我找了好多打印模块的解决方案，但各有利弊，这些模块都是别人设计好的，别想太多创新性，从头造轮子的工作，我们对此只能适应。


以上是做软件开发实习的经历，仅供学弟学妹们参考，我还是希望同学们选择互联网公司工作，因为那样你会从身边的人上学到东西而不断成长。如果不想朝九晚五的生活，不想脱离美妙的数学，还想探索知识的边界，那就读研究生吧，或者项目够牛，就创业吧，接外包吧！如果你问我的话，我兼而有之，还记得听北大的一次公选课，雷鸣建议北大的学生要在技术的前沿创业，一般技术水平的创业，你想到了，有可能别人在这类项目上已经淹死了只是你不知道而已。我选择读博士，也是尝试在技术前沿寻找可能的商机。

### 6.谈谈AI公司面试
接下来说一下前两天到旷世科技面试的经历。我是由北大实验室的师兄内推前往面试的，为的是找份暑期实习工作，赚点零花钱同时也可以学一些东西，这些公司有专门的科研岗位的，在做科研的同时他们给你发工资。我头一天修改了份简约的简历，只保留了几个重量级的奖项，发送给师兄后，师兄很快就通知我那边面试官安排我第二天下午2点面试了。和面试官加了微信，面试官和我说起再过不久就要到美国盐湖城参加CVPR会议，这些年这些AI创业公司都喜欢发顶会，商汤等同样如此。第二日我提前到达，在搜狐网络大夏的一楼坐着休息。当时我身边坐着一个老板加一个员工，老板50岁不到，员工30岁不到，老板语气很硬，偶出脏话，而那员工（青年男子）小心谨慎，礼貌有加，更多时间低着头，声音很弱。我听到员工说这个项目遇到了一些困难，老板说：“屁，人从生到死就是不断遇到困难的过程。。。你不能干，给你调行政岗，那个困难少。。。给你3天时间梳理一下，我要看书面报告。。。”我后面和另一个朋友分享了这件事，朋友问我的想法，我只是回答：那个青年男子一定要有一个贤惠体贴的妻子啊！事业不顺真的需要有一个人理解。。。这是个小插曲，分享给大家。我准时到达旷世的那个楼层，这些初创公司都选择落座在中关村一带，中关村寸土寸金，一栋大楼往往有数家知名互联网公司，即使是旷世科技所在楼层，我也看到了另一家牙科公司。我在等面试官的时候，看了看旷世的宣传视频等，高科技的含量十足，让人觉得科技时代真美好，但我知道其中的一些行业瓶颈，没有被某些宣传手段误导，我自认为还是有批判性精神的啦。


面试官带我沿着两侧全是没有隔断，紧密排列的工位进到最里面的一间小会议室，在走的过程中，听到有人当着我的面暗地说中国石油大学不行的等等。我很镇定，心想：“你NB，不也同样打工，即使是旷世科技，商业模式也很单一，也就人脸识别做的还好，而你们做的是别的方向，发两篇水文，有什么值得炫耀的！”。我觉得风度还是要有，不卑不亢嘛，面试官首先让我自我介绍，然后便挑着简历上的内容问我问题，他不太关注数学建模等的奖项，换做是我，我也不会关注，哈哈，这东西你有我有，大家都有。而他关注我做的与深度学习相关的内容，我介绍了所做的3个深度学习研究经历。他问我做的这个“人群密度估计”在哪些数据集上做的，效果怎么样，什么样的网络，他让我画出来，不拘一格的面试，直接拿笔在玻璃覆盖的桌子上画。他居然关注我做机器人的经历，他问起中国机器人行业发展如何？我们想象中的机器人是否可以马上到来？他也表示了对类人机器人的兴趣。我还是有些意外的，他没有问我与视觉检测和网络压缩相关的工作，这是他们组所做的核心工作啊，亏我准备了一些。他也问了我有没有参加算法类比赛，我说起蓝桥杯的省二奖项，不知道他知不知道这个比赛，可能在他们这些清北高校出来的，只知道ACM吧！他后来出去了，叫了一个胖胖的程序员进来要考我算法题，我有些叫苦不迭，不是说好了深度学习面吗？这也太不拘一格了！第一题贪心，第二题数组的一个题，直接在玻璃上写代码，我开始意识到《剑指offer》那类图书的重要性。以上是面试AI公司的经历，希望给学弟学妹们一些参考。

### 7.谈谈保研面试
我参加过中科院自动化所和北京大学信科院的夏令营，收到了中科院硕士和北大直博的offer并选择了北大。自动化所的夏令营组织的很好，我们所关心的面试，笔试都穿插在各种娱乐活动，扩展活动中间。我看到了来自天津大学，东北大学，北航，华南理工，合肥工大等高校的不同专业同学，他们给我的感觉是这样的，大部分还是更会考试，有很多男生，理工科男生一贯的那种腼腆他们都有，但我同时也看到了他们普通样貌下的各种才艺与才华。我想说说刘和静，与我一同过去面试的我们班的一位女生，她成功当选自己组的组长并在最后的文艺汇演上担任女主持人，各种积极的发言，一如她的性格，我由衷地钦佩其能力，她似乎具有全场所有学霸都不具备的那种技能，让我意识到石油大学的社团还是给了我们非常丰富的锻炼机会的，而她最后成功进入自动化所可能也和这有关系。从一个合肥工大的数学专业的学生那里，我了解到他对很多数学课的用处不了解，为了学习而学习，我甚至从计算机的纯应用的角度给他讲了几个应用领域。一个华南理工的学生，专业课有60多分的，但是组队参加了国际上的某算法类竞赛，据说排名第2，自动化所排名第3，其对深度学习相关的知识非常自信，同时也有到地平线机器人创业公司做实习生。至于面试阶段，前面共坐着10几位研究员，自我介绍后便是项目经历，这些老师不看重开发，但是看中算法，与AI相关的东西，一位老师看到我简历上对FPGA了解问我有没有做过FPGA写神经网络。数学题目问了道什么是拉格朗日中值定理，我的回到被评价为没有用数学的严谨性语言阐述。

接下来聊聊北大的面试，简短的英文自我介绍后，同样要抽题目，我抽了道什么是线性空间，以及解释二分查找算法。后面便是聊项目，我了解到自己还是有项目优势的，其它学校的学生在这方面同样匮乏。这些老师喜欢搞过机器人的学生，一位老师还问我能不能给她做一个可以控制开机的机器人，哈哈！一个老师有打趣地问我愿不愿意跟他一起创业，我可以听得出来是有一些开玩笑在里面的。聊项目的时候，一个老师直接和我说，让我挑做的最好的项目先说，这意思就是说最好的项目如果不行，其它的也不用看了，我也意识到一定要优化完善一个项目，我们说的工匠精神还是要有的，我还是有点朝秦暮楚的性格的，做事不够专心。聊天的逻辑很重要，这个需要在大学期间锻炼，我和我现在的导师第一次聊，我发挥的便很好，我的导师非常注重学生解决问题的能力，他会问你做的这个项目的难点在哪里，你在面对这个问题的时候是怎么解决的。畅想一个将近60岁的科研人员，一辈子都是不断地面对问题，解决问题，他必然关注学生的这一素质。我从我导师的言语中可以感受到思维的高度抽象性，全面性，他同时也是不迷信任何权威的，他甚至在一次组会上专门探讨所有知识技能的共性，我们对某一个事物感兴趣，从科研的角度而言即使是外行，投入足够时间和经历能不能够做好？值得一说的是我曾跟着徐朝农老师做一个病床角度测量的单片机项目，现在想想那个项目还可以用卡尔曼滤波方法进行优化，当我说起这个项目时，导师认为这是一个软件项目，我之前一直以为是硬件呢！什么是硬件相关的呢？我说起了机器人比赛的经历，导师问了我机器人几个自由度等相关的问题。他同时也问了我有没有学过信号处理和人工智能机器学习相关的课程，我这时候体味到选修朱丹丹老师机器学习的必要性；复变函数，最优化，数字图像处理，数字信号处理等课程自己有自学了解过的英明。


### 8.希望大家关注的一些东西
- 大家可以写自己的csdn博客，知乎专栏，简书甚至是github代码库，这在面试中值得一说，同时对你的知识整理有很大帮助。
- 学习linux相关命令操作，vim操作（安装自己喜欢的配置）等，我现在身边的师兄师姐都是在linux上运行程序的。
- 学习git版本控制工具（建议），或者svn
